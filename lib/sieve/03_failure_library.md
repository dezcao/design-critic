# 실패 라이브러리 — Sieve 판정 케이스

> 이 문서는 비넬리 캐넌을 체(Sieve)로 사용했을 때 **얕은 판단과 올바른 판단이 갈리는 사례**를 수록한다.
> 각 사례는 "표면적으로 개선된 것처럼 보이지만 실제로는 후퇴한" 패턴을 중심으로 한다.
> 체커(평가자)가 빠지기 쉬운 칭찬을 먼저 기록하고, 그것이 왜 얕은지를 명시한다.

---

## 사례: HYUDO 리브랜딩 (AI 피드백 → 외부모델 재디자인)
- 입력: 얇은 라인아트 원형 심볼(원본). 피드백 = 폐쇄형 원 전환, 획 1.5배, 수면 단서(달·물결) 삽입, 4종 버전 정비.
- 표면 결과: 집 + 초승달 + 물결의 폐쇄형 원 심볼. 두꺼운 선·정렬·보조버전 보드까지 = '완성도 신호' 충족.
- 얕은 판단(피해야 할 평가): "업종 가독성↑, 견고성↑, 버전 정비됨 → 개선됨."
- 올바른 판단: 명사 3개(집·달·물결)를 한 원에 욱여넣은 덧셈형 일러스트. 네모+삼각지붕은 인류 공통 '집' 픽토그램이라 독자성 0이며, '주택/부동산'을 가리켜 업종을 *틀리게* 신호한다. 원본의 프리미엄 톤을 잃고 더 흔한 자리로 후퇴 = 가독성을 얻고 독자성을 잃음.
- 매핑되는 원칙: [가독성 ≠ 직설적 묘사] / [덧셈은 냄새 — 요소 추가는 거의 항상 후퇴] / ['업종 추론 가능성'은 무효 지표 — 헤스텐스·캐스퍼·템퍼는 제품을 그리지 않는다] / [마감 신호 ≠ 디자인 품질].

## 사례: 동네 고깃집 간판의 5색 그라데이션 리뉴얼
- 입력: 흰 배경에 검정 고딕 텍스트만 있는 단색 간판(원본). 피드백 = "너무 밋밋하다, 식욕을 자극하는 색을 넣어라."
- 표면 결과: 빨강·주황·노랑·금색 그라데이션 배경 위에 흰 글자 + 불꽃 아이콘 + 고기 이모지. 포화도 최대.
- 얕은 판단(피해야 할 평가): "식욕 자극 색상 사용, 정육점 업종 연상, 에너지 있어 보임 → 업종 적합성 개선됨."
- 올바른 판단: 빨강·주황·노랑의 동시 사용은 서로의 힘을 상쇄한다. 어느 색도 지배하지 못하므로 결과적으로 무채색화된 혼탁함이 된다. 불꽃 아이콘과 고기 이모지는 의미 중복이며 모두 '직설적 묘사'의 덧셈이다. 원본의 흑백 대비는 야간·원거리·흑백 인쇄에서 모두 읽혔지만, 그라데이션 버전은 실제 간판 출력 시 야간에서 글자가 배경에 녹아든다. 색을 늘려 힘을 잃었다.
- 매핑되는 원칙: [색채 — 색은 관계와 비례로 작동한다, 많을수록 무채색화된다] / [시각적 힘 — 강렬함은 복잡성이 아닌 구조적 명확성에서 온다] / [빛 — 빛의 조건을 고려하지 않은 색채 선택] / [자의성의 거부 — "식욕 자극"은 이유가 되지 못한다, 왜 이 다섯 가지 색인가에 답할 수 없다].

## 사례: 법무법인 로고의 트렌디한 그라디언트 리브랜딩
- 입력: 짙은 네이비 + 세리프 서체 + 방패 형태의 로고(원본, 창립 20년). 피드백 = "너무 올드하다, 요즘 로펌들처럼 현대적으로 바꿔라."
- 표면 결과: 퍼플-블루 그라디언트 심볼 + 얇은 산세리프 워드마크. 깔끔하고 스타트업처럼 보임.
- 얕은 판단(피해야 할 평가): "트렌디하고 세련됨, 젊어 보임, 접근성 있어 보임 → 이미지 개선됨."
- 올바른 판단: 20년간 축적된 브랜드 자산(신뢰, 권위, 역사성)을 완전히 폐기했다. 새 디자인이 전달하는 코드('스타트업', '테크')는 법무법인이 전달해야 할 코드('신뢰', '안정', '전문성')와 정반대다. 트렌드가 지나면 3년 후 다시 '촌스러워 보이는' 사이클에 진입한다. 원본의 세리프+네이비는 영구성을 가졌으나, 그라디언트는 특정 연도를 박제한 타임스탬프가 된다. 클라이언트가 "현대적으로"를 원했을 때 실제 필요(needs)는 '더 접근하기 쉬운 이미지'였지, 업종 코드의 전환이 아니었다.
- 매핑되는 원칙: [브랜드 자산(Equity) — 오랜 자산은 정제해야지 폐기하면 안 된다] / [영구성·내구성 — 그라디언트는 트렌드 타임스탬프다] / [적합성 — 형태가 업종 맥락을 이탈했다] / [실용론 — 클라이언트의 wants와 needs를 구분하지 못했다].

## 사례: 스타트업 명함의 5종 서체 혼용
- 입력: 이름·직함·연락처·회사명·슬로건이 각각 다른 서체로 조판된 명함 초안. 피드백 = "개성 있어 보이고 다양한 느낌을 주고 싶다."
- 표면 결과: 이름은 핸드라이팅 서체, 직함은 두꺼운 산세리프, 회사명은 세리프, 연락처는 모노스페이스, 슬로건은 이탤릭 스크립트. 5개 서체가 한 명함에 공존.
- 얕은 판단(피해야 할 평가): "정보가 모두 포함되어 있고, 계층이 느껴지며, 개성이 있어 보임 → 완성도 있음."
- 올바른 판단: 서체가 5개라는 것은 시각적 규칙이 5개라는 뜻이다. 규칙이 5개면 규칙이 없는 것과 같다. 각 서체는 서로 다른 "문법"을 말하고 있어 보는 사람이 우선순위를 결정할 수 없다. "개성"이라 부르는 것은 실제로는 구문론적 무정부 상태다. 동일 서체 패밀리 안에서 크기·굵기·자간만으로도 충분한 위계와 개성이 만들어진다. 서체를 늘릴수록 개성이 아니라 혼란이 축적된다.
- 매핑되는 원칙: [타이포그래피 — 소수의 서체로 충분하다, 서체 증식은 시각적 오염이다] / [구문론 — 요소 간 관계가 일관된 규칙 아래 있어야 한다] / [스케일과 위계 — 서체 변경이 아닌 크기·굵기로 위계를 만들어야 한다] / [자의성의 거부 — "개성 있어 보이고 싶다"는 이유가 서체 선택의 근거가 되지 못한다].

## 사례: 어린이 영어학원 로고의 '선생님처럼 보이게' 수정
- 입력: 밝은 노랑·초록의 아이콘 + 동그란 서체로 구성된 원본 로고. 피드백 = "너무 유치해 보인다, 전문성 있어 보이게 바꿔라."
- 표면 결과: 짙은 남색 배경 + 금색 장식 테두리 + 세리프 서체 워드마크. 대학 엠블럼처럼 보임.
- 얕은 판단(피해야 할 평가): "전문성과 신뢰감이 생겼고, 고급스럽게 보임 → 업그레이드됨."
- 올바른 판단: 이 로고의 의사소통 대상은 어린이와 학부모 중 어린이에게 '가고 싶은 곳'으로 신호해야 한다. 짙은 남색+금색 엠블럼은 어린이에게 '무서운 곳', '재미없는 곳'으로 읽힌다. 학부모에게만 최적화하면서 실제 사용자(어린이)를 배제했다. '전문성'이라는 피드백이 실제 필요(need)인지, 아니면 원장의 자아 표현 욕구(want)인지 구분하지 못한 결과다. 원본의 밝은 색채와 동그란 형태는 유치한 것이 아니라 맥락에 적합한 것이었다.
- 매핑되는 원칙: [적합성 — 형태가 타겟 맥락이 아닌 제작자 욕구에 봉사했다] / [의미론 — 업종(어린이)이 전달해야 할 신호와 형태가 충돌한다] / [실용론 — 클라이언트의 wants(전문성)와 실제 needs(어린이 유입)를 구분하지 못했다] / [문맥 — 어린이 교육기관의 문맥을 금융기관 문맥으로 교체했다].

## 사례: 동네 제과점의 '프리미엄화' 간판 교체
- 입력: 빨간 배경에 흰 글자, 창업자 이름을 딴 30년 된 간판(원본). 피드백 = "고급 베이커리처럼 보이게 바꿔라."
- 표면 결과: 크림색 배경 + 연한 골드 얇은 서체 + 영문 이름 + 밀 이삭 일러스트. 파리 빵집 분위기.
- 얕은 판단(피해야 할 평가): "고급스럽고 세련됨, 트렌디한 베이커리 느낌, 인스타그래머블 → 리브랜딩 성공."
- 올바른 판단: 30년 단골들이 가게를 알아보지 못하게 됐다. 원본의 빨간 배경+흰 글자는 강한 대비로 원거리에서도 읽혔으나, 크림색+골드 얇은 서체는 야외 자연광에서 거의 읽히지 않는다. '파리 빵집' 코드는 이 동네 중장년 단골들에게 '낯선 곳', '비쌀 것 같은 곳'으로 읽힌다. 축적된 지역 사회 인지 자산을 폐기한 것이며, 새로운 고객층(인스타 세대)을 유입하기 위한 전략이 기존 고객층 이탈 비용을 상회하는지 검토하지 않았다. '고급스러워 보임'은 빛 조건에서의 가독성과 상충했다.
- 매핑되는 원칙: [브랜드 자산(Equity) — 30년 지역 자산을 폐기했다] / [빛 — 크림+골드 얇은 서체는 야외 자연광에서 읽히지 않는다] / [문맥 — 실제 고객(동네 단골)의 맥락을 무시하고 가상의 고객(인스타 세대)에 맞췄다] / [적합성 — '고급 베이커리' 코드가 이 지역·이 고객 맥락에서 적합한가를 묻지 않았다].

## 사례: 공공기관 웹사이트 메뉴판의 '미니멀리즘' 리디자인
- 입력: 정보가 밀도 있게 배치된 구형 웹사이트 메뉴 구조(원본). 피드백 = "복잡해 보인다, 미니멀하게 정리해라."
- 표면 결과: 흰 배경 + 넓은 여백 + 연한 회색 텍스트 + 아이콘만 남긴 햄버거 메뉴. 깔끔하고 현대적.
- 얕은 판단(피해야 할 평가): "깔끔하고 미니멀함, 여백이 충분하고 현대적 UI 패턴 적용 → 사용성 개선됨."
- 올바른 판단: 이 공공기관의 주 사용자(중장년·노년층)는 아이콘만 있는 햄버거 메뉴를 '메뉴'로 인식하지 못한다. 연한 회색 텍스트는 명도 대비가 WCAG 기준에 미달하여 저시력자에게 읽히지 않는다. '미니멀'이라는 피드백이 정보 제거와 동의어로 처리되었고, 실제로는 접근성과 기능을 삭제했다. 여백은 '콘텐츠와 맥락에 최적으로'가 기준인데, 이 경우 사용자가 정보를 빠르게 탐색해야 하는 실용적 맥락을 무시하고 시각적 고급함을 선택했다. 미니멀리즘이 기능 삭제의 면죄부가 되었다.
- 매핑되는 원칙: [실용론 — 기능이 먼저, 형태는 그 다음이다] / [여백 — 여백은 '최대한 많이'가 아니라 '맥락에 최적으로'다] / [책임 — 노인·저시력자를 배제하는 색 대비는 디자이너의 윤리적 책임 위반이다] / [문맥 — 화면에서 아름다운 것이 실제 사용자 맥락에서는 실패한다].

## 사례: 카페 로고의 '복잡한 빈티지 엠블럼' 채택
- 입력: 원형 텍스트 + 중앙 커피컵 + 장식 리본 + 설립연도 + 별 장식으로 구성된 신규 로고 시안.
- 표면 결과: 빈티지 배지 형태, 세밀한 일러스트, 다수의 장식 요소. "장인 정신"과 "전통"을 연상시킴.
- 얕은 판단(피해야 할 평가): "디테일이 풍부하고 고급스럽다, 장인 카페 분위기 연출됨, 완성도 높은 일러스트 → 브랜드 개성 있음."
- 올바른 판단: 이 로고는 앱 아이콘(64×64px)으로 축소하면 아무것도 읽히지 않는 검은 덩어리가 된다. 자수, 각인, 흑백 영수증 인쇄에서도 마찬가지다. '복잡한 빈티지 엠블럼'은 복수의 맥락에서 작동하지 않는다. 설립연도를 넣는 행위는 실제 역사가 없을 때 역사를 연출하는 것으로, 의미론적 위조다. 장식 요소 하나하나가 "왜 있는가"에 답할 수 없다면 그것은 노이즈다. 세밀한 일러스트의 완성도가 시스템 작동 가능성과는 무관하다는 것을 간과했다.
- 매핑되는 원칙: [실용론 — 다양한 매체에서 실제로 작동하는가] / [문맥 — 화면 시안과 실제 적용 환경(앱 아이콘, 각인, 흑백 인쇄)의 문맥 차이를 무시했다] / [의미론 — 없는 역사를 연출하는 것은 의미론적 불일치다] / [지적 우아함 — 설명할 수 없는 장식은 노이즈다, 완성도 있는 노이즈도 노이즈다].
