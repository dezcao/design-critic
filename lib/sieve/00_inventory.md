# 인벤토리 — The Vignelli Canon (한국어판)

## 기본 정보

- **총 페이지 수**: 120페이지 (스캔 기준)
- **원본 파일**: `the-vignelli-canon.pdf` (~80MB)
- **형식**: 스캔 이미지 PDF — 텍스트 레이어 없음
- **스캔 장비**: ApeosPort-III C5500 (Fuji Xerox 복합기)
